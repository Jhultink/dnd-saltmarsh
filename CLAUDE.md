# Saltmarsh D&D Campaign

This workspace contains no code — only markdown files tracking a D&D campaign set in and around the port town of Saltmarsh.

## Workspace Structure

- [characters/](characters/) — One markdown file per player character and notable NPC
- [sessions/](sessions/) — Session recap notes, one file per session

## Working in This Repo

- All content is markdown. Keep formatting consistent and human-readable.
- Character files live in `characters/`. Name them `firstname-lastname.md` or just `firstname.md`.
- Session files live in `sessions/`. Name them `session-##-short-title.md` (e.g., `session-01-arrival-in-saltmarsh.md`).
- Commit after each session recap or major update so history serves as a campaign log.

## Campaign Notes

- **System:** D&D 5e
- **Setting:** Ghosts of Saltmarsh (official module + homebrew extensions)
- **Town:** Saltmarsh — a fishing town on the Azure Sea with a complicated relationship with smugglers, sea elves, and the lizardfolk of the nearby marshes.
