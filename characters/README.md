# Characters

This folder contains markdown files for player characters and notable NPCs.

## File Naming

- Player characters: `pc-firstname.md` or `pc-firstname-lastname.md`
- NPCs: `npc-name.md`
- Enemies/factions: `faction-name.md`

## Character File Template

When creating a new character file, use this structure:

```markdown
# Character Name

**Player:** (for PCs only)
**Race:** 
**Class/Level:** 
**Background:** 
**Alignment:** 

## Description

Brief physical description and personality.

## Backstory

Key backstory beats relevant to the campaign.

## Goals & Motivations

What does this character want?

## Relationships

Connections to other PCs, NPCs, or factions.

## Notes

Ongoing campaign notes — secrets learned, oaths made, items of significance, etc.
```
